Discord Emoji v1.0 (Android)

A random app idea that came into my mind when I was looking for custom emojis for my discord server. I found Developer API from discordemoji.net. At that time, there was no App available for this service, so by using Sketchware I made this app in about 3-4 days and decided to keep it ad-free.
Features of This App:
  • Download Any Emoji
  • View Details of particular emote
  • Categorise the large list and find the best emoji
  • Download Emoji Packs
